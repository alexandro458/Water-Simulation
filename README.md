# Simple Water Rendering

by Alejandro Vega


## Features

* Sum Of Sines Fluid Simulation (from Catlike Coding)
* * Sine Wave
* * Sum of sinus
* * Gerstner Waves
* Albedo fluid animation
* * UVs animation
* * Texture distortion and looping
* Basic transparency
* * Water fog + Fog color
* * Underwater Refraction

*Shaders or scripts are not optimized, just demo ready*

## What's Missing?

* Buoyancy
* Atmospheric shader
